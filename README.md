# 📘 Eduniti – Study Material & Learning Platform

**Eduniti** is a modern, beginner-friendly **study material and coding tutorial website** designed to provide structured learning resources for students and aspiring developers.
The platform combines clean UI design with interactive functionality to make learning more engaging and accessible.

🔗 **Live Demo:** [https://code-domain.github.io/Eduniti/](https://code-domain.github.io/Eduniti/)

---

## ✨ Key Highlights

* 📚 **General Zone** with academic resources

  * Notes
  * Books (PDFs)
  * PYQs (Previous Year Questions)
  * Tutorials

* 💻 **Coder Section** featuring programming tutorials:

  * HTML
  * CSS
  * JavaScript
  * Python
  * C
  * C++
  * Java
  * React JS
  * PHP

* 🎯 Interactive buttons like **Start Learning** and **Want Notes**

* 📱 Fully responsive layout for different screen sizes

* 🎨 Clean, modern dark-themed UI

* ⚡ Smooth navigation and user interaction

---

## 🛠️ Tech Stack

* **HTML5** – Page structure and content
* **CSS3** – Styling, layout, and responsiveness
* **JavaScript** – Interactivity and DOM manipulation
* **JSON** – Dynamic data handling for study materials

## 🎯 Project Objective

The main goals of Eduniti are:

* To practice **real-world front-end development**
* To understand **UI/UX design principles**
* To work with **dynamic data using JSON**
* To build an educational platform that is simple yet scalable
